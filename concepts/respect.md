# Respect

1. Respect is a reputation score expressing positive opinion towards individual community member from the perspective of the whole community;
2. Respect is non-transferrable;
3. Respect scores are determined using a [credibly neutral](https://balajis.com/p/credible-neutrality) democratic consensus process run by the community;
4. Respect is not a property that can by owned by anyone (or anything) and no participating party owns Respect;
